# 🌾 Analyzing Agriculture Productivity Across Indian States

This Power BI project provides an in-depth analysis of crop production trends across India using interactive dashboards and visualizations. By exploring 19 years of historical data from 33 states and 646 districts for 122 unique crops over five seasons, this report uncovers key insights into agricultural productivity across regions and time.

---

## 📊 Project Overview

**Dashboards Included:**

1. **📌 Overview and Insights**
   - Total production metrics
   - Season and year-wise production changes
   - Top crops and top-performing states and districts
   - Crop production by season
   - Season filter for dynamic exploration

2. **📌 Detailed Analysis**
   - Crop diversity index trends
   - State-wise crop production map
   - Pie chart showing average production by area
   - Line chart visualizing total production and cultivated area by year
   - Filter options for state-wise breakdown

---

## 🧠 Key Insights

- **Kerala** ranks as the **top producing state**.
- **Coconut** is the **most produced crop** across the dataset.
- Season-wise analysis shows the **Whole Year** has the highest yield.
- Districts like **Kozhikode** and **Malappuram** lead in total production.
- Regional disparities highlight potential for **growth through crop diversification**.
- Data suggests strong **seasonal variations** and historical trends worth considering for forecasting.
